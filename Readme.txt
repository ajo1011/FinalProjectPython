How to Run the Quiz Maker Program:

1. Make sure you have Python installed on your computer. You can download it from https://www.python.org/downloads/.
   Note: This program requires Python 3.x.

2. Download the "quiz_maker.py" file and the "testbank.txt" file and save them in the same directory.

3. Open a terminal or command prompt and navigate to the directory where you saved the files.

4. Run the program by typing the following command and pressing Enter:
   python quiz_maker.py

5. Follow the prompts to enter your information, choose the number of questions for the quiz (10 or 20), and answer the questions.

6. After completing the quiz, the program will display your score and elapsed time, and it will generate a text file with your quiz results.

7. You can run the program again to take another quiz or exit the program by typing "Q" when prompted.

Enjoy the quiz!
